- [Raw WebGPU](https://alain.xyz/blog/raw-webgpu) - A WebGPU tutorial
- [WebGPU Fundamentals](https://webgpufundamentals.org/) - Walkthroughs

 - [WebGL 2 Specification](https://registry.khronos.org/webgl/specs/latest/2.0/)
