This demo downloads GPT-2 124M model and tokenizer artifacts from Andrej Karpathy's
[llmc-starter-pack](https://huggingface.co/datasets/karpathy/llmc-starter-pack), which
packages files for the MIT-licensed [llm.c](https://github.com/karpathy/llm.c) workflow.
GPT-2 was originally released by OpenAI in the
[gpt-2](https://github.com/openai/gpt-2) repository.
